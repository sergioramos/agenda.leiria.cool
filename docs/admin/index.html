Passo 1 · GitHub

Ligação ao GitHub

É isto que dá poder aos botões deste painel. O token fica guardado apenas neste navegador. Permissões necessárias: Actions, Contents e Secrets (Read and write).

Passo 2 · Inteligência artificial

Chave DeepSeek

A recolha usa a DeepSeek para ler as páginas dos locais. Cole aqui a sua chave (criada em platform.deepseek.com) — é cifrada neste navegador e guardada como segredo do repositório. Nunca fica visível.
